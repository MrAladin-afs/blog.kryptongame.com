**Microsoft is planning a free, ad-supported tier for Xbox Cloud Gaming** (xCloud), according to internal reports. Sources confirm that the company has been testing this new option with employees, which will allow users to stream games without a costly Game Pass subscription.

This free service will reportedly feature **pre-roll ads** lasting around two minutes before a game starts. Early testing suggests playtime will be **limited to one hour per session** and about five hours total per month, though these details could change before public release. This move aims to make Xbox game streaming more accessible.

* **Free Streaming Coming:** Microsoft is reportedly close to announcing a free, **ad-supported tier** for Xbox Cloud Gaming (xCloud).
* **No Game Pass Required:** This option will allow users to stream Xbox games **without a Game Pass subscription**.
* **Ad Model:** The service will feature **pre-roll ads** reportedly lasting about two minutes before the game loads.
* **Play Limits:** Internal testing suggests **time limits** will apply: one hour per session and a maximum of five hours per month. (These limits are subject to change.)
